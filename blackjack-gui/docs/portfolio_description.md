# ポートフォリオ説明メモ

## 作品名

Blackjack Item Battle

## 作品概要

Pythonで制作した，アイテム要素付きブラックジャックGUIアプリです．
通常のブラックジャック操作に加えて，手札交換，2択カード選択，ラウンド再開始，敗北回避などの独自アイテムを実装しています．

## 制作意図

コンソール上で動作していたゲームを，GUIアプリとして遊べる形に改修しました．
改修時には，表示処理とゲームロジックが混在しないように，`game.py` と `gui.py` に責務を分離しました．

## 技術的に工夫した点

- TkinterによるGUI実装
- `BlackjackGame` クラスによるゲーム状態の管理
- ゲームロジックとUI処理の分離
- 状態に応じたボタン制御
- テストしやすいスコア計算・アイテム処理の設計
